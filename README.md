🐉 Three Body Resonance Lab

Interactive three-body dynamics lab for exploring stability, chaos, and orbital resonance:

How do three gravitational bodies behave — and can they ever settle into stable, repeatable motion?

This project is not a solver.
It’s a sandbox for discovery.

🎯 What this is

A real-time, browser-based simulation where you can:

Adjust mass, gravity, distance, and velocity
Observe how three bodies interact over time
Explore transitions between:
Stable motion
Chaotic scattering
Binary capture
Emergent structure
🧠 What makes this different

Instead of just visualizing motion, this system measures behavior:

📊 Core Metrics
Stability score (bounded vs unstable)
Triangle geometry quality
Coherence (combined structural signal)
Center-of-mass drift
Energy + angular momentum
🔁 Resonance Detection

The sim looks for repeating timing relationships between bodies:

Identifies approximate orbital ratios (e.g. 2:3 ≈ 5:4)
Scores how strongly motion aligns
Flags resonant behavior
🏆 Gold State Detection

When a configuration meets strong conditions:

Stable (bounded, low drift)
High coherence
Strong resonance
Sustained over time

The system flags:

Stable Configuration Found

This is not proof of a general solution —
but it highlights interesting candidate states worth exploring.

🎮 How to use
Start with the Triangle preset
Keep masses equal
Set moderate gravity
Slowly adjust velocity

Or…

Hit Randomize and explore.

You’ll see the system move through:

Chaotic scatter
Binary capture
Distorted triangles
Structure forming
Resonant locking
🔬 What to look for
Bodies staying bounded over time
Repeating orbital paths
Low drift in center of mass
Simple timing ratios (resonance)
Triangle structures tightening or relaxing
⚠️ What this is NOT
Not a closed-form solution to the three-body problem
Not a physics engine for precision modeling
Not making claims about new physical laws

This is a tool for intuition and exploration

🧭 Why this exists

The three-body problem is known for its sensitivity to initial conditions.

Rather than solving it analytically, this project lets you:

interact with the system directly and observe behavior emerge
